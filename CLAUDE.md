# NotchyPrompter — project guidance for Claude

Native macOS Swift/SwiftPM app. System audio → WhisperKit → Claude/Ollama → notch overlay.

## Critical gotchas
- **TCC resets on signature change.** `./build.sh` ad-hoc signs every time, which invalidates the Screen Recording grant. Do not rebuild mid-session unless the user expects to re-grant permission.
- **`excludesCurrentProcessAudio = false` in `AudioCapture.swift:48` is intentional.** Temporarily relaxed for debugging; do not "fix" unless explicitly asked (tracked in CHANGELOG).
- **No tests exist.** Do not run `swift test` and report results — there are no targets.
- **Per-callback debug logs in `Pipeline.swift` and `AudioCapture.swift` are on the cleanup list.** Don't add more.

## Build & run
- Build: `cd NotchyPrompter && ./build.sh` (produces `NotchyPrompter.app` with ad-hoc sign). Do not run `swift build` alone — TCC keys off the bundle.
- Launch: `open NotchyPrompter/NotchyPrompter.app`
- Kill: `pkill -f NotchyPrompter`
- Output location: `NotchyPrompter/NotchyPrompter.app/Contents/MacOS/NotchyPrompter`

## Architecture contracts
- Audio chain: `SCStream` → `AVAudioConverter` (16 kHz mono Float) → `VADChunker` → `Transcriber` (WhisperKit) → `LLMClient` → `OverlayViewModel`.
- New LLM backends conform to `LLMClient` (see `ClaudeClient.swift`, `OllamaClient.swift`). Claude: SSE + `anthropic-beta: prompt-caching-2024-07-31`. Ollama: NDJSON with `think: false`.
- `Pipeline` is `@MainActor`; `AudioCapture` is `@unchecked Sendable` and callbacks run on `.global(qos: .userInitiated)` — cross the actor boundary explicitly.
- Swift 6 strict concurrency is on.

## Defaults (don't change silently)
- Claude model: `claude-haiku-4-5-20251001`
- Ollama URL: `http://localhost:11434`
- Whisper model: `openai_whisper-large-v3-v20240930_turbo`
- VAD: RMS threshold 0.01, 800 ms speech, 400 ms silence, 15 s hard cap
- Claude prompt cache: ephemeral, ~5 min TTL

## Don't touch
- `NotchyPrompter/.build/**`, `NotchyPrompter/.swiftpm/**`
- `NotchyPrompter/NotchyPrompter.app/Contents/**` (rebuilt by `build.sh`)
- `Package.resolved` (committed, regenerated by SwiftPM)
